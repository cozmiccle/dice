# Dyce Roller
a fun little tool designed for MacOS Tahoe

## Description
ever struggled to just pick something? dyce is here to help! just pick one of the many options (with an added kanji/chinese mode) and roll away!
simplify everything from where to eat dinner to what color to put your kid's college fund on on a fun night out!

## Usage
installation is easy, simply download the zip file containing the version you'd like from the releases page and drag it into your applications folder on your Mac.

if you want to build the program yourself, download xcode developer tools from wherever you get it nowadays, clone the repo into a folder, `cd` to that folder, and run
```zsh
xcodebuild -scheme dice
```
the built app should be spit out into `build/`
